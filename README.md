# 🌱 Plant Care — Aplikacija za zalijevanje biljki

Završni projekt iz predmeta **Programiranje mobilnih uređaja**.  
Aplikacija omogućuje ručno upravljanje zalijevanjem biljki  
pomoću **ESP32** mikrokontrolera i Android mobilne aplikacije.

## 📱 O projektu

Sustav se sastoji od **Android aplikacije** i **ESP32 uređaja**  
koji međusobno komuniciraju kako bi korisnik mogao ručno  
pokrenuti zalijevanje biljke putem mobitela.

## ⚙️ Kako radi?

1. Korisnik otvori **Android aplikaciju**
2. Pritisne gumb za **zalijevanje**
3. Aplikacija šalje naredbu na **ESP32**
4. ESP32 aktivira **LED** kao simulaciju pumpe za vodu
